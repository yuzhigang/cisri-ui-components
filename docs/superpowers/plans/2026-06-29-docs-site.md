# Docs Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single-page documentation and live-demo site (`apps/docs`) that consumes `@cisri/json-schema-editor` from the workspace and demonstrates default, dark, and custom-themed usage.

**Architecture:** A Vite + React app inside the pnpm workspace. It imports the local `@cisri/json-schema-editor` package, shares the root Tailwind config, and renders interactive demos alongside install/usage/theming docs. No backend is required.

**Tech Stack:** React 18, TypeScript, Vite, Tailwind CSS, React Router (or in-page sections), lucide-react for nav icons.

---

## File Structure

- `apps/docs/package.json` — workspace package definition, depends on `@cisri/json-schema-editor` and peer deps.
- `apps/docs/vite.config.ts` — Vite dev server config, alias `@cisri/json-schema-editor` to its source or dist as needed for HMR.
- `apps/docs/tsconfig.json` — extends `tsconfig.base.json`, references `@cisri/json-schema-editor`.
- `apps/docs/index.html` — HTML entry point.
- `apps/docs/src/main.tsx` — React root render.
- `apps/docs/src/App.tsx` — top-level layout with sidebar navigation and theme switcher.
- `apps/docs/src/index.css` — CSS variable definitions for default + dark themes and custom theme example.
- `apps/docs/src/sections/HomeSection.tsx` — project overview and quick start.
- `apps/docs/src/sections/InstallSection.tsx` — install command and peer dependency list.
- `apps/docs/src/sections/DemoSection.tsx` — live `JsonSchemaEditor` demo with state preview.
- `apps/docs/src/sections/ThemingSection.tsx` — CSS variable table and live theme toggle buttons.

- [ ] **Step 3: Create `apps/docs/src/components/ThemeToggle.tsx`**

```tsx
import { useEffect, useState } from 'react';
import { Moon, Sun, Waves } from 'lucide-react';

type Theme = 'default' | 'dark' | 'ocean';

const themes: { id: Theme; label: string; icon: typeof Sun }[] = [
  { id: 'default', label: 'Light', icon: Sun },
  { id: 'dark', label: 'Dark', icon: Moon },
  { id: 'ocean', label: 'Ocean', icon: Waves },
];

export function ThemeToggle() {
  const [theme, setTheme] = useState<Theme>('default');

  useEffect(() => {
    const root = document.documentElement;
    root.classList.remove('dark', 'theme-ocean');
    if (theme === 'dark') root.classList.add('dark');
    if (theme === 'ocean') root.classList.add('theme-ocean');
  }, [theme]);

  return (
    <div className="flex items-center gap-2 rounded-md border border-border bg-card p-1">
      {themes.map(({ id, label, icon: Icon }) => (
        <button
          key={id}
          onClick={() => setTheme(id)}
          className={`flex items-center gap-1.5 rounded-sm px-2.5 py-1.5 text-xs font-medium transition-colors ${
            theme === id
              ? 'bg-primary text-primary-foreground'
              : 'text-muted-foreground hover:bg-accent hover:text-accent-foreground'
          }`}
        >
          <Icon className="h-3.5 w-3.5" />
          {label}
        </button>
      ))}
    </div>
  );
}
```
